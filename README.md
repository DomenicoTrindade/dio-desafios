# 🚀 Desafios e Projetos - DIO

Repositório criado para armazenar exercícios, desafios e projetos desenvolvidos durante os cursos e bootcamps da plataforma Digital Innovation One (DIO).

O objetivo deste repositório é registrar minha evolução prática em programação, versionamento de código e desenvolvimento de software.

---

## 📚 Conteúdo

Este repositório reúne exercícios, desafios e projetos desenvolvidos durante meus estudos na plataforma **Digital Innovation One (DIO)**.

Os conteúdos abordados incluem:

- Lógica de programação
- Controle de versão com Git e GitHub
- Desenvolvimento em Python
- Desenvolvimento em Java
- Fundamentos de banco de dados
- Projetos práticos aplicados
